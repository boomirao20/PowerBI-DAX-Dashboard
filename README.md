## 📊 Dataset Details

This project uses multiple datasets to simulate a real-world business scenario:

* **customers.csv** → Customer details
* **orders.csv** → Order information
* **products.csv** → Product data
* **sales.csv** → Transaction-level sales data

### 🔗 Relationships

* Customers → Orders (customer_id)
* Orders → Sales (order_id)
* Products → Sales (product_id)

---

## 🚀 Key Features

* 📈 Revenue trend analysis
* 👥 Customer insights
* 🛍️ Product contribution to sales
* 💰 Profit and cost calculation
