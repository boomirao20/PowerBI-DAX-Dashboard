# 📊 Power BI Dashboard with DAX

## 👩‍💻 Author

Boomi Rao

## 📌 Project Description

This project demonstrates how to build an interactive Power BI dashboard using DAX (Data Analysis Expressions).

It covers:

* Data Import
* Data Transformation
* Data Modeling (Relationships)
* DAX Measures
* Data Visualization
